Uma ferramenta de **OSINT (Open Source Intelligence)** iniciante desenvolvida em Python para coleta e análise de informações públicas de domínios, IPs e redes.

---

# Funcionalidades:

- 🌐 DNS Lookup (resolução de domínio)
- 🧾 WHOIS Lookup (informações do domínio)
- 📍 Geolocalização de IP
- 🔍 HTTP Headers Analysis
- 🔓 Port Scanner (portas comuns abertas)
- 📊 Geração de relatórios em JSON
- 📱 Informações sobre número de telefone

---

# Tecnologias usadas:

- Python 3
- socket
- requests
- python-whois
- pyfiglet
- dnspython
- OS module
- phonenumbers
- pycountry
